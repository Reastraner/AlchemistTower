# Alchemist Tower

Небольшая консольная текстовая игра на C#.

Игрок приходит в себя в запертой темнице и пытается выбраться наружу, исследуя пещеру, башню и её шпиль. По мере прохождения герой узнаёт правду о себе, Алхимике и повторяющемся цикле событий.

## Особенности

- Несколько игровых локаций
- Исследование помещений
- Инвентарь и предметы
- Диалог с Алхимиком
- Две концовки
- Выбор действий через консольное меню

## Управление

Введите номер нужного действия и нажмите Enter.

- `0` — выйти в главное меню
- `9` — открыть инвентарь
- Остальные варианты зависят от текущей локации

## Запуск

Для запуска необходим установленный .NET SDK.

1. Склонируйте репозиторий
2. Откройте проект в Visual Studio
3. Запустите приложение

Или выполните в папке проекта:

```bash
dotnet run
```

## Структура проекта

- `Game` — основной игровой цикл и логика локаций
- `MainMenu` — главное меню
- `Player` — данные игрока
- `Inventory` — хранение предметов
- `Item` — описание предметов
- `Room` — данные игровых помещений
- `Location` — список доступных локаций
